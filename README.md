## 🖥️ What this app does

py-prompt-versioner helps you keep AI prompts organized in a simple versioned flow. It is made for people who work with prompts, Markdown files, and command-line tools.

You can use it to:
- store prompts in a clear file structure
- track prompt changes over time
- check prompt content before use
- work with Markdown prompts
- manage prompt files with a simple CLI flow
